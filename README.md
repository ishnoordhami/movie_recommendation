# Movie Recommendation System Using KNN

A Movie Recommendation System based on movie ratings using K-Nearest Neighbors (KNN) and Collaborative Filtering.

## Features
- Movie recommendations based on user ratings
- KNN algorithm
- Cosine similarity
- Streamlit web application

## Dataset
MovieLens Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run Locally

pip install -r requirements.txt

streamlit run app.py
